# CudaVM

> :warning: **Experimental software**: Use at your own risk!

A CUDA-based bytecode interpreter for massively parallel smart contract execution.

## Setup

`nvcc` should be installed and accessible in `$PATH`.

```
# build
make

# run
make run
```
